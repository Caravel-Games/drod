Player Selection Screen

Choose "Change Player" on the Main Menu to access the Player Selection Screen. This screen has the following options available:

Okay - selects the highlighted player profile. Each player profile contains user preference settings and its own game progress.

New Player - creates a new player profile. This option is useful when you have more than one person playing DROD on this installation.

Delete player - permanently deletes the selected player profile. This includes deleting the player's saved games, so make sure this is really what you want to do!

Export - saves the active player profile to disk with all its saved games. This option facilitates transferring your player settings and game information to a different game installation.

Import - loads a player profile exported to disk. This allows you to recover exported player saved games. Hint: to import saved games for a specific hold (set of levels), you must first import the hold into your installation, and then import the player data.

Cancel - returns to the previous screen.
