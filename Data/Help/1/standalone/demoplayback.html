Demo playback controls:

  1. Escape - End demo playback.
  2. Move E/W command or Left/Right Arrow - Seek to the previous/next move. The longer you hold the button the faster it'll go.
  3. Move N/S command or Up/Down Arrow - Increase or reduce playback speed.
  4. Numbers 0-9 - Seek to the 10% to the 100% of the demo.
  5. Show stats command or Enter - Go to the specific move in the demo.
  6. Restart command - Seek to the beginning of the demo.
  7. Skip Cutscene command or Space - Pause/Unpase the demo playback.
  8. Move NW command - Toggle uniform demo playback.

The screen will not close on its own at the end of the demo playback, you must press Escape to go back to the previous screen. If this is a multi-room demo it'll automatically continue to the next demo and there is no way to go back to the previous room.
